> Simple User Managment Server for adding tasks and reminders for team members or as a basic task list with system logs

## Prerequisites

- npm >=5.5.0
- node >=9.3.0

## Usage

```sh
node app.js
OR
Run "Start_Server.bat" (Windows only)
```
Hosts local server at ```http://127.0.0.1:3000```
